# TotoStory TV WordPress Theme

토토스토리 리뉴얼 시안을 기반으로 만든 워드프레스 테마 초안입니다.

## 포함된 화면

- 홈: TV 포털형 메인 화면
- 기존 워드프레스 전면 페이지 본문 자동 출력
- 홈 하단 최신 글 6개, 기존 주요 페이지 12개 노출
- 기존 워드프레스 메뉴 자동 유지
- 글 목록/카테고리/글 상세
- `중계 일정` 커스텀 포스트 타입
- `검증 리포트` 커스텀 포스트 타입
- `먹튀 제보 접수` 페이지 템플릿
- 공식/허가된 링크와 임베드 URL 입력용 관리자 필드

## 설치

1. 워드프레스 관리자 > 외모 > 테마 > 새로 추가 > 테마 업로드
2. `totostory-tv-theme.zip` 업로드
3. 테마 활성화
4. 설정 > 고유주소에서 저장 버튼 클릭
5. 기존 메뉴가 바로 안 보이면 외모 > 메뉴에서 기존 메뉴를 `Primary Menu`에 배정

## GitHub + Cloudways 자동 배포

GitHub 저장소와 Cloudways SSH 배포를 연결하려면 `CLOUDWAYS_DEPLOY.md`를 따라 설정하세요.

## 운영 메모

- 실제 영상 스트림은 포함되어 있지 않습니다.
- `중계 일정` 글에서 공식/허가된 링크 URL 또는 허가된 임베드 URL을 직접 입력하세요.
- `먹튀 제보 접수` 페이지는 전송 기능이 없는 시안 폼입니다. 실제 운영 시 폼 플러그인, 텔레그램 알림, 또는 별도 DB와 연결하세요.
- 기존 워드프레스 글과 카테고리는 기본 글 목록/상세 템플릿으로 표시됩니다.
- 기존 전면 페이지의 본문 내용은 홈 중간 `기존 전면 콘텐츠` 섹션에 표시됩니다.
- 기존 메뉴가 새 테마 위치에 배정되지 않아도, 테마가 워드프레스에 남아 있는 첫 번째 메뉴를 자동으로 사용합니다.
- 홈 상단 빠른 목록과 하단 주요 페이지는 `토토스토리 스포츠tv`, `스포츠 다시보기`, `팀순위 확인`, `경기결과`, `라이브스코어`, `먹튀제보` 같은 기존 페이지 제목을 우선 찾아서 표시합니다.
